A Simple 3-Day Workout Planner

Description: I chose this name because it is concise, memorable, and immediately hints at the core content‚Äîa structured, limited number of workouts/exercises (3 days, 4 exercises per day). It sounds strong and professional for a fitness resource.

üéØ Site Purpose

The primary purpose of A Simple 3-Day Workout Planner is to provide a simple, structured, and easily accessible three-day fitness plan - Upper Body, Lower Body, Core for beginners and intermediate users.

The site serves as a crucial demonstration of responsive web design and core JavaScript functionality, specifically DOM manipulation and event handling.

Information and Services Provided:

  • Clear outlines for three distinct workout days with 4-5 exercises each.
  • A JavaScript-driven Difficulty Toggle on the Upper Body page (Beginner/Intermediate sets/reps).
  • A tool to dynamically add personalized notes next to core exercises on the Core page.

3. ü§î Scenarios (Target Audience Questions)

These questions represent the target audience's needs and drive the site's content and functionality.

  1. "I'm a beginner, How do I know which sets and reps to use for the Upper Body workout?"

    Answered by: The Difficulty Toggle" button on the Upper Body page, which uses JavaScript event handling to instantly switch the displayed set/rep values between Beginner (e.g., 3 sets of 10) and Intermediate (e.g., 4 sets of 12).

  2. "I want to track my progress and need to remember a modification for one of the Core exercises. Is there a place I can write that down?"

    Answered by: The Add Notes" button on the Core page, which uses JavaScript to dynamically insert a new, empty text input field (modifying the HTML DOM) where the user can type personalized notes.
